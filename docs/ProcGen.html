(inner) makeFloatMap(sources, fn) → {FloatBuffer}

Creates a new floating-point texture, optionally based on existing textures. This type of texture can store any single floating-point value for each pixel.
Parameters:
Name Type Description
sources Array the array of sources this texture will be based on
fn function the function that will be called to determine the value at every point. It will be called with the value of all the sources at this point, and the X and Y coordinates of the point. It should return the resulting floating point value.

(inner) simplexNoise(seed, octaveCount, roughness) → {buffer}

Returns a floating-point buffer with simplex noise. Values for the noise will be in the range [-1.0,1.0].
Parameters:
Name Type Description
seed number a random seed (different values produce different noises)
octaveCount integer the number of octaves of noise to sum together - the more octaves the finer detail you will have
roughness number how fast the noise will change. 1.0 is a safe default, higher number result in rougher, denser noise, lower numbers result in smooth, gentle noise.
